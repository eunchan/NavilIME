나빌 입력기를 빌드하는 방법
====================
나빌 입력기는 윈도우에서 비주얼 스튜디오를 이용해 빌드할 수 있습니다. 비주얼 스튜디오로 NavilIME.sln 파일을 열어서 Release 옵션을 선택하여 빌드 합니다. 

개발을 목적으로 빌드하는 것이라면 Debug 옵션으로도 빌드가 됩니다. 이 때는 Global.h 파일의 DEBUG_LOGFILE 설정을 수정해야 로그 파일이 제대로 만들어집니다.




나빌 입력기를 인스톨하는 방법
=====================
나빌 입력기를 빌드하여 dll 파일이 생성되면 수동으로 아래 경로의 폴더를 만듭니다.

C:\Program Files\NavilIME

그리고 관리자 권한으로 윈도우 커맨드를 실행합니다. 관리자 권한의 윈도우 커맨드에서 install.cmd를 실행하면 인스톨이 완료됩니다.




나빌 입력기를 제거하는 방법
====================
관리자 권한으로 윈도우 커맨드를 실행합니다. 그리고 uninstall.cmd를 실행하면 나빌 입력기가 시스템에서 제거됩니다. 그러나 아직 파일은 남아 있으므로 C:\Program Files\NavilIME 폴더를 수동으로 삭제해 주세요.




나빌 입력기의 키보드 설정을 변경하는 방법
==============================
C:\Program Files\NavilIME\Keyboard.dat 파일을 수정합니다.

아래의 이름중 하나를 위 파일에 적습니다.

318Na
2
2y
32
39
3f
3s
3y
ro

반드시 한 줄로 위 목록 중 한 개를 하나만 적어야 합니다. 기본값은 318Na입니다. 예를 들어 두벌식으로 바꾸고 싶다면 설정 파일의 내용을 아래와 같이 바꿉니다.

2

딱 저렇게 한 글자만 적으면 됩니다. 마찬가지로 세벌식 3-90 자판을 사용하고 싶다면 아래와 같이 설정 파일의 내용을 바꿉니다.

39

각 설정에 대한 설명은 아래 내용을 참조하세요.

318Na 	: 	세벌식 3-18Na 자판
2			:	두벌식
2y			:	두벌식 옛한글
32			:	세벌식 두벌 레이아웃
39			: 	세벌식 3-90 자판
3f			: 	세벌식 3-91 (최종) 자판
3s			: 	세벌식 순아래
3y			: 	세벌식 옛한글
ro			: 	한글 로마자

설정 파일을 수정하고 나면 입력기의 원활한 사용을 위해 시스템을 재부팅해 주세요.
